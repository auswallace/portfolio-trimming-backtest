# How to Create PDF from HTML

The HTML file `Taking_Profits_What_Actually_Works.html` contains all 8 charts embedded.

## Method 1: Print to PDF from Browser (Easiest)

1. Open `Taking_Profits_What_Actually_Works.html` in Chrome or Edge
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows)
3. Select "Save as PDF" as destination
4. **Important Settings:**
   - Background graphics: **ON**
   - Margins: Default
   - Scale: 100%
5. Click "Save"
6. Name it: `Taking_Profits_What_Actually_Works.pdf`

**Result:** Professional PDF with all charts embedded, ready to share!
